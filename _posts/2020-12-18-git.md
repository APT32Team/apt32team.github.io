Test thử đánh tiếng Việt có dấu
Các câu lệnh git thường dùng như bên dưới
Có thể tạo lệnh Git tự động đẩy code bằng tập hợp các dòng lệnh bên dưới:
Bỏ tất cả cách lệnh bên dưới vào một file tên: deploy.sh

$ git status

$ git add .

& git commit -a -m "first commit""

``` $ git push ```

Sau đó save lại
Tiếp theo, cấp quyền run cho file bằng lệnh
```chmox +x deploy.sh```
